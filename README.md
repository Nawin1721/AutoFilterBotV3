# 🤖 AutoFilterBotV3

A powerful Telegram Auto Filter Bot built using Python, MongoDB, and Telegram Bot API.

---

## ✨ Features

- 🔍 Smart Movie Search
- 📄 Pagination Buttons
- 🌐 Language Filters
- 🎥 Quality Filters
- ⭐ IMDb Integration
- 🧹 Auto Delete Messages
- 🔐 Force Subscribe System
- ⚡ Fast Search Results
- 📂 MongoDB File Storage

---

## 🛠 Technologies Used

- Python
- python-telegram-bot
- MongoDB
- PyMongo
- RapidFuzz
- Telegram Bot API

---

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/Nawin1721/AutoFilterBotV3.git

Install Requirements
pip install -r requirements.txt

⚙️ Environment Variables

Create environment variables:
BOT_TOKEN=your_bot_token
API_ID=your_api_id
API_HASH=your_api_hash
MONGO_URI=your_mongodb_uri
GROUP_ID=your_group_id
DB_CHANNEL_ID=your_channel_id
FORCE_SUB_CHANNEL=your_channel_username

🚀 Run Bot 
python main.py

🌍 Deploy
Supports:
Render
Koyeb
Railway
